## Methodology

Source priority used here: official paper/arXiv, official project repo/model zoo/docs, official model cards, reputable benchmark papers or implementation repos, then clearly labeled third-party evidence. A value is `Confirmed` only when the cited source exposes the same metric in the same or clearly compatible benchmark context. A value is `Corrected` when the cited source reports a different value. `Conflict` means sources report different values because dataset, training recipe, runtime backend, or evaluation protocol differs. `Unsupported` means I could not find source text for the claim in the checked sources. `NR fill` means the original cell was `NR`, but source-backed information can be added if the main report wants that context.
